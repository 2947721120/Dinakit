Gettings StartedAll you need to know about Dinakit Framework

To install DinaKit Framework simply download our latest release and link both CSS and JS files.

Alerts Alerts are used to display all types of messages : error, success, warning or info.

To use the Alert message simple create an element such us a div and give it the class class="alert". The default alert message is the warning version.


Default Alert

Element : <div class="alert">...</div>

Error Alert

Element : <div class="alert alert-error">...</div>

Info Alert

Element : <div class="alert alert-info">...</div>

Success Alert

Element : <div class="alert alert-success">...</div>


Alerts can be closed by adding an a element with as class alert-close.

The element we use : <span class="alert-close"></span>

Breadcrumb

Breadcrumbs are always a nice feature to have on a website. With DinaKit, it is so easy to use theme.

Simply create a class="breadcrumb" parent inside which you put a <class="item"> children containing your <a> elements. Active items should have active class.
